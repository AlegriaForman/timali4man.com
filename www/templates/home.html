About

This website is intended to be a hybrid of a personal site along with our professional portfolios. We hope it can be a place for friends and family to visit along with a place we where can continue to hone our web dev skills.

Slightly technical stuff

We wanted to upgrade our old website after gaining more experience working with web technologies, especially AngularJS. Using a 'mobile first' strategy we selected the Ionic framework to get started with. Being familiar with AngularJS helped in learning to use Ionic and while any framework can be a bit top-heavy we were pleased with all that can be done with Ionic and it's many built in features.

Slightly more technical

In addition to the main three (HTML, CSS, JavaScript) and Ionic the site also uses NodeJS and Gulp. It may seem unusual to use Ionic as the framework of a site for both mobile devices and desktop PCs but we found that it can easily handle both just fine.

This site is hosted by MDD Hosting and Cyberduck was used for uploading via FTP.
